# Biscuits v2.0

Using the Laravel web framework written in PHP.

## Setup

### Clone the repository

```bash
git clone https://github.com/GarvJain03/biscuits-v2.0.git
```

### Install Dependecies

```bash
composer install
```

### Fill out .env

```bash
cp .env.example .env
```

#### Add Webhook Url in .env (For Discord notifications)

`DISCORD_WEBHOOK_URL`

### Run App

```bash
php artisan serve
```
